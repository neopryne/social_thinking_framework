Exists, though we can never know we have found it.

It exists by the following collapse argument:
There are two possibilities:
* "Absolute truth exists" is a true statement.
* "Absolute truth exists" is not a true statement.

We can't be sure of it because it is impossible for a mind to be certain that it knows anything.
